\# ResumeIQ



ResumeIQ is a web application that helps users analyze and improve their resumes.



\## Features

\- Resume parsing

\- Skill matching

\- Score generation



\## Tech Stack

\- Java

\- Spring Boot

\- HTML, CSS

\- GitHub



\## How to Run

1\. Clone the repo

2\. Import into IDE

3\. Run the application
